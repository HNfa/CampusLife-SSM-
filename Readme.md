### 项目名称
校园生活(SSM)
*****
### 项目功能
基于SSM开发，功能类似于美团，覆盖校园周边吃喝玩乐，  
分为客户端、商户端、管理端
*****
### 运行环境
Mysql8.0、Jdk1.8、Redis4.0、Tomcat8  
Chrome浏览器
*****
### 项目部署
1. 配置jdbc.properties、redis.properties
2. 配置PathUtil.java文件，确定图片存储路径
